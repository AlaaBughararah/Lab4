This repository contains my Lab 4 assignment on Data Quality Assessment and Preprocessing.

The following tasks were completed:
- Data quality assessment (data types, structure, summary)
- Data cleaning (fixing Date and Amount formats)
- Handling missing values using mean, median, and mode
- Detecting and removing outliers using the IQR method
- Applying normalization techniques (Min-Max and Z-score)
- Correlation analysis between numerical features
- PCA analysis (evaluated but not required due to weak correlation)

Tools used: Python, Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn.
